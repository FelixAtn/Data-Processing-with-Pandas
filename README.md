# Data Processing with Pandas

Python project demonstrating practical data analysis and CSV manipulation using pandas.

Created as part of university coursework focused on database and big data concepts.

## Features

* Load and manage multiple CSV datasets
* Filter and sort records
* Add calculated columns
* Handle missing values
* Compute averages and summaries
* Export transformed datasets

## Tech Stack

* Python
* pandas
* Jupyter Notebook

## Purpose

Demonstrates data handling, transformation, and analytical workflows using Python.
